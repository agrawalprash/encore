HTTP StoresÂ¶

This module contains several different stores that communicate with a remote HTTP server which provides the actual data storage.

The simplest of these is the HTTPStore which can be run against a static HTTP server which provides a json file with all metadata and then serves data from URLS from another path. It polls the metadata periodically for updates.
